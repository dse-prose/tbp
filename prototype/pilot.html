Use the prototype

This is a manually coded prototype for the Parallel view for Chapter 4 of Prose's Third Book.

The prototype is meant to show key features the Parallel view, which is designed to support comparison between the text of the princeps and the other witnesses. For these reason, though visible on the interface, the Single View mode and the Lists are not available.

The buttons at the top allow to select the comparison between V and P or between P and the subsequent printed editions.

In the text box dedicated to manuscript Vat.lat.3210, the user can select a textual reconstruction from the dropdown at the top, which offers three possibilities: base draft, revision and final text.

Other features available in the prototype include:

  • Paragraphemic variants, which can be shown or hidden using the toggle button in the footer
  • Named Entities (at the current stage, only people mentioned in the dialogue), which can be highlighted via a button at the bottom of the page. Clicking on the name of the person displays tooltips with contextual information. The tooltip can be closed by clicking again on the name
  • Quotations, which are highlighted in yellow on hover. Clicking on a quotation opens a modal with the corresponding bibliographic reference.
  • A Legend, accessible via the bottom-right button, summarizes the chromatic and typographic markers, as detailed in the DSE mockup section of the website.

    • Additionally, green asterisks (*) have been used to signal omographs; a tooltip on hover shows information about the word.

    Finally, for documentation purposes only, passages involved in substantial variants are provided with a label, which records the TEI elements processed for the textual reconstruction.
